-<h2 align="center" id="usage">Usage</h2>	
-	
-<pre align="center"><code>cash &lt;amount&gt; &lt;currency code&gt;	
-</code></pre>	
-	
-<p align="center">for example:</p>	
-	
-<pre align="center"><code>cash 1 usd	
-</code></pre>	
-	
-<p align="center">or</p>	
-	
-<pre align="center"><code>cash 10 usd eur rub pln	
-</code></pre>
